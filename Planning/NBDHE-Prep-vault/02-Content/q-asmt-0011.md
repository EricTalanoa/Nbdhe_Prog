# Stem
A patient has a Class II molar relationship with retroclined (lingually inclined) maxillary
central incisors and a deep overbite. This occlusal relationship is classified as:

# Options
- A) Class II, Division 1
- B) Class II, Division 2 (correct)
- C) Class III, Division 1
- D) Class I with a bimaxillary protrusion

# Correct answer
B

# Rationale
Class II malocclusion is subdivided by maxillary incisor position: Division 1 shows protrusive
(flared, labially inclined) maxillary incisors, while Division 2 shows retroclined (lingually
inclined) maxillary central incisors, often with a deep overbite. The retroclined presentation
described here specifically identifies Division 2.

**Why the distractors are wrong**
- A) Division 1 describes protruded, flared incisors — the opposite of the retroclined
  presentation given.
- C) Class III involves a mesial molar relationship (mandibular first molar positioned mesially
  relative to the maxillary first molar), an entirely different molar relationship than the Class
  II relationship described.
- D) This describes a Class I (normal molar relationship) case with both arches flared forward,
  not the Class II relationship stated in the stem.

# Stem
All of the following statements about irreversible hydrocolloid (alginate) impression material are
true EXCEPT:

# Options
- A) The set material results from an irreversible chemical (gelation) reaction between sodium or potassium alginate and calcium sulfate.
- B) Once set, the material cannot be returned to a fluid state by reheating.
- C) Increasing the water-to-powder ratio shortens the material's working and setting time. (correct)
- D) The impression should be poured promptly, since the material is prone to syneresis (water loss) and imbibition (water uptake) if stored improperly.

# Correct answer
C

# Rationale
Increasing the water-to-powder ratio dilutes the reacting components and actually LENGTHENS
(not shortens) alginate's working and setting time. This is the false statement — a common trap,
since a more dilute mix intuitively seems like it should react faster.

**Why the distractors are wrong**
- A) True — this irreversible chemical reaction is why alginate is called "irreversible"
  hydrocolloid, unlike agar (reversible hydrocolloid).
- B) True — once the calcium alginate gel network forms, heat cannot re-liquefy it, unlike agar.
- D) True — alginate impressions must be poured promptly and wrapped in a damp towel if delay is
  unavoidable, to minimize dimensional change from syneresis and imbibition.

# Stem
The most reliable way to confirm that a steam autoclave is actually achieving sterilization is:

# Options
- A) a biological (spore) test using resistant bacterial spores (correct)
- B) the color change on the autoclave indicator tape
- C) confirming the programmed cycle time has elapsed
- D) verifying the instruments look visibly clean afterward

# Correct answer
A

# Rationale
Biological indicators contain highly heat-resistant spores (e.g., *Geobacillus stearothermophilus*).
If a cycle kills them, it has met the conditions lethal to essentially all microorganisms — the only
method that directly verifies sterilization. It is performed at least weekly.

**Why the distractors are wrong**
- B) Indicator tape is a chemical process indicator: it shows a package was exposed to heat, not
  that sterilizing conditions were achieved.
- C) Elapsed time alone does not confirm that temperature and pressure were adequate throughout.
- D) Visible cleanliness reflects instrument processing, not microbial sterility.

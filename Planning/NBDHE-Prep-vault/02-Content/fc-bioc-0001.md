# Front
Why doesn't xylitol contribute to caries the way sucrose does?

# Back
Xylitol is a 5-carbon sugar alcohol that cariogenic bacteria (*S. mutans*) **can't ferment**
through normal glycolysis — no acid end-product forms, so plaque pH doesn't drop. Some strains
still take it up via a fructose transporter into an energy-wasting "futile cycle," which over time
can *reduce* cariogenic bacterial counts. Contrast with sucrose, which ferments readily and drives
the classic Stephan-curve pH drop below the ~5.5 critical pH.

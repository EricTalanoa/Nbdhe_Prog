# Stem
A study reports p = 0.03 for the difference in mean probing depth reduction between two
scaling techniques, using an alpha of 0.05 set in advance. What does this p-value indicate?

# Options
- A) There is a 3% probability that the null hypothesis is true.
- B) There is a 97% probability that the observed difference is clinically important.
- C) If the null hypothesis were true, a difference this large or larger would occur about 3% of the time by chance alone (correct)
- D) The alternative hypothesis has been proven correct.

# Correct answer
C

# Rationale
The p-value is the probability of observing a result as extreme as, or more extreme than, the one
obtained, assuming the null hypothesis is true. A p-value of 0.03 falling below the pre-set alpha
of 0.05 leads the researcher to reject the null hypothesis, but the p-value itself is a statement
about the data given the null, not a probability statement about the null (or alternative)
hypothesis, and it says nothing about clinical significance.

**Why the distractors are wrong**
- A) The p-value does not give the probability that the null hypothesis is true; that
  misinterpretation reverses what conditional probability the p-value describes.
- B) Statistical significance (p < alpha) is distinct from clinical importance; a small, clinically
  trivial difference can still be statistically significant with a large enough sample.
- D) Hypothesis testing never "proves" the alternative hypothesis; a low p-value only provides
  evidence against the null hypothesis within the stated significance level.

# Stem
Clinical attachment level (CAL) is measured from the:

# Options
- A) gingival margin to the base of the pocket
- B) cementoenamel junction to the base of the pocket (correct)
- C) mucogingival junction to the gingival margin
- D) gingival margin to the mucogingival junction

# Correct answer
B

# Rationale
CAL is measured from a fixed anatomic landmark — the cementoenamel junction — to the base of the
sulcus or pocket. Because the reference point does not move, CAL reflects true attachment loss even
when the gingival margin has receded or is swollen.

**Why the distractors are wrong**
- A) That describes probing depth, which is measured from the (movable) gingival margin and can
  mislead when there is recession or edema.
- C) That describes the width of keratinized/attached tissue, not attachment level.
- D) The gingival-margin-to-mucogingival-junction distance also relates to gingival width, not
  attachment.

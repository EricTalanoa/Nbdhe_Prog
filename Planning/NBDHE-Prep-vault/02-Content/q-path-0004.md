# Stem
A patient develops hives, facial swelling, and difficulty breathing within minutes of receiving an
injection of amide local anesthetic containing a sulfite preservative. This reaction is mediated by
which hypersensitivity mechanism?

# Options
- A) Type I — IgE-mediated (correct)
- B) Type II — cytotoxic antibody-mediated
- C) Type III — immune complex-mediated
- D) Type IV — delayed, T-cell-mediated

# Correct answer
A

# Rationale
Type I hypersensitivity is an immediate reaction in which allergen-specific IgE bound to mast cells
and basophils triggers degranulation and release of histamine and other mediators on re-exposure,
producing urticaria, angioedema, and bronchospasm/anaphylaxis within minutes — matching this
patient's rapid-onset presentation.

**Why the distractors are wrong**
- B) Type II reactions involve antibodies (IgG/IgM) directed against antigens on the patient's own
  cell surfaces (e.g., hemolytic reactions), not a rapid systemic allergic response to a drug.
- C) Type III reactions involve antigen-antibody immune complexes depositing in tissue (e.g., serum
  sickness), which develops over days, not minutes.
- D) Type IV reactions are cell-mediated and delayed, typically appearing 24–72 hours after
  exposure (e.g., allergic contact dermatitis to latex), not within minutes.

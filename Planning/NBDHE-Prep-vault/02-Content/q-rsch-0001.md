# Stem
In a data set of resting heart rates that contains one extreme outlier, which measure of central
tendency is most resistant to that outlier?

# Options
- A) mean
- B) median (correct)
- C) range
- D) standard deviation

# Correct answer
B

# Rationale
The median is the middle value by rank, so a single extreme value barely shifts it. This makes the
median a robust (resistant) measure of central tendency when the data are skewed or contain outliers.

**Why the distractors are wrong**
- A) The mean is pulled toward extreme values and is therefore sensitive to outliers.
- C) The range is a measure of spread, not central tendency, and is defined by the extremes.
- D) The standard deviation is a measure of spread and is inflated by values far from the mean.

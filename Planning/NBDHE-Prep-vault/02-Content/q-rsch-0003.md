# Stem
A screening test with high sensitivity is especially good at:

# Options
- A) correctly identifying people who truly have the disease, with few false negatives (correct)
- B) correctly identifying people who are truly disease-free
- C) confirming a diagnosis once disease is already strongly suspected
- D) producing the same result when the test is repeated

# Correct answer
A

# Rationale
Sensitivity = true positives / (true positives + false negatives). A highly sensitive test rarely
misses true disease, so a negative result is useful for ruling the condition out (the "SnNout"
concept).

**Why the distractors are wrong**
- B) Correctly identifying disease-free people describes specificity, not sensitivity.
- C) Confirming disease when suspected relates to specificity and positive predictive value.
- D) Reproducibility of a repeated test is reliability, a different property from sensitivity.

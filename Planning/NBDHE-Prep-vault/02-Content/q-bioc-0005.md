# Stem
Which salivary buffer system is primarily responsible for neutralizing acid at LOW (unstimulated,
resting) salivary flow rates?

# Options
- A) Bicarbonate buffer system
- B) Phosphate buffer system (correct)
- C) Protein/mucin buffer system
- D) Urea buffer system

# Correct answer
B

# Rationale
Bicarbonate concentration in saliva is strongly flow-rate dependent, rising sharply only once flow
is stimulated (e.g., by chewing or eating); at rest it contributes relatively little. The phosphate
buffer system's concentration changes much less with flow rate, making it the dominant buffer at
low, unstimulated flow. Once flow is stimulated, bicarbonate becomes the dominant buffering system.

**Why the distractors are wrong**
- A) Bicarbonate is actually the dominant buffer at HIGH, stimulated flow rates, not at rest —
  the reverse of what this question asks.
- C) Salivary proteins and mucins contribute some buffering, but only a minor amount compared with
  phosphate at resting flow.
- D) Urea is a minor substrate that bacteria can convert to ammonia in plaque; it is not a primary
  component of saliva's own buffering capacity at rest.

# Front
What are the three Spaulding classification categories for reprocessing patient-care items, and
what does each require?

# Back
Critical (penetrates soft tissue/bone, e.g. scalers) → sterilize. Semicritical (contacts mucous
membranes but doesn't penetrate tissue, e.g. mirrors, impression trays) → at minimum high-level
disinfection, sterilization preferred. Noncritical (contacts only intact skin, e.g. BP cuff,
chair surfaces) → low- to intermediate-level disinfection.

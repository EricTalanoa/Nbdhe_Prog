# Stem
The four classic cardinal signs of acute inflammation are redness, heat, swelling, and:

# Options
- A) pain (correct)
- B) suppuration
- C) necrosis
- D) fever

# Correct answer
A

# Rationale
The four classic cardinal signs described by Celsus are rubor (redness), calor (heat), tumor
(swelling), and dolor (pain). Loss of function (functio laesa) was added later by Virchow.

**Why the distractors are wrong**
- B) Suppuration (pus formation) may accompany inflammation but is not one of the four classic signs.
- C) Necrosis is tissue death, a possible consequence, not a cardinal sign.
- D) Fever is a systemic manifestation, not one of the four local cardinal signs.

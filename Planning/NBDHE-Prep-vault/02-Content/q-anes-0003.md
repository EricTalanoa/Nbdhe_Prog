# Stem
Adding epinephrine to a local anesthetic solution provides all of the following advantages EXCEPT:

# Options
- A) prolonged duration of anesthesia
- B) reduced systemic absorption of the anesthetic
- C) decreased bleeding at the injection site
- D) a faster onset because the solution is more alkaline (correct)

# Correct answer
D

# Rationale
This is a negative-format item — the key is the statement that is NOT true. Epinephrine-containing
solutions are actually more *acidic* (they include an antioxidant such as sodium metabisulfite),
which tends to slow onset slightly. So "faster onset because the solution is more alkaline" is both
false and not an advantage.

**Why the distractors are wrong**
- A) Vasoconstriction slows removal of the anesthetic, genuinely prolonging duration — a real
  advantage, so it is not the exception.
- B) Slowed absorption lowers peak plasma levels and improves safety — a real advantage.
- C) Local vasoconstriction reduces bleeding in the operative field — a real advantage.

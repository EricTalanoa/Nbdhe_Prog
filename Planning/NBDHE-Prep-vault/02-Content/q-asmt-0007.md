# Stem
In the Glickman classification of furcation involvement, a Class II furcation is best described
as bone loss that:

# Options
- A) has not yet entered the furcation, only a groove is palpable
- B) enters the furcation as a cul-de-sac but is not through-and-through (correct)
- C) passes completely through the furcation but is still covered by soft tissue
- D) passes completely through the furcation and is visible clinically due to recession

# Correct answer
B

# Rationale
Class II involvement means the interradicular bone has been lost enough that an explorer or
probe enters the furcation area, but a portion of bone remains so the lesion does not
communicate all the way through to the opposite side (a "cul-de-sac").

**Why the distractors are wrong**
- A) That describes Class I (incipient), where only a groove or concavity is felt and bone loss
  has not yet entered the furcation.
- C) That describes Class III, a through-and-through lesion still hidden by the interdental
  papilla/gingiva.
- D) That describes Class IV, a through-and-through lesion that is visible directly because the
  gingiva has receded away from it.

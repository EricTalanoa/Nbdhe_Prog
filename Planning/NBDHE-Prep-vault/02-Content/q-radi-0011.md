# Stem
All of the following statements about vertical angulation errors on a periapical radiograph are
true EXCEPT:

# Options
- A) Excessive (too steep) vertical angulation causes foreshortening, making the tooth image appear shorter than its actual length.
- B) Insufficient (too flat) vertical angulation causes elongation, making the tooth image appear longer than its actual length.
- C) Correcting a foreshortened image on the retake requires increasing the vertical angulation. (correct)
- D) Correcting an elongated image on the retake requires increasing the vertical angulation.

# Correct answer
C

# Rationale
Foreshortening is caused by excessive vertical angulation, so correcting it on the retake requires
DECREASING vertical angulation, not increasing it — increasing it further would only worsen the
foreshortening. This is the false statement.

**Why the distractors are wrong**
- A) True — this is the standard cause of foreshortening.
- B) True — insufficient angulation under-projects the image, elongating it.
- D) True — since elongation comes from too little angulation, increasing angulation on the
  retake corrects it.

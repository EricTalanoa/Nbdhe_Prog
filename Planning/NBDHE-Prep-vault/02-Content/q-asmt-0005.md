# Stem
In an Angle Class I molar relationship, the mesiobuccal cusp of the maxillary first molar occludes
with the:

# Options
- A) buccal groove of the mandibular first molar (correct)
- B) mesiobuccal cusp of the mandibular first molar
- C) space between the mandibular second premolar and first molar
- D) distobuccal cusp of the mandibular second molar

# Correct answer
A

# Rationale
Angle Class I ("neutrocclusion") is defined by the mesiobuccal cusp of the maxillary first molar
occluding in the buccal (mesiobuccal) groove of the mandibular first molar. Class II and Class III
describe posterior and anterior deviations from this reference, respectively.

**Why the distractors are wrong**
- B) A cusp-to-cusp relationship of the maxillary and mandibular mesiobuccal cusps is not the
  Class I landmark.
- C) A relationship one tooth mesial (toward the premolar) reflects a Class II tendency, not
  Class I.
- D) The distobuccal cusp of the mandibular second molar is not the defining Class I contact point.

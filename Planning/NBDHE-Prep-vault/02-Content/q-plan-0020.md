# Stem
During case presentation, a hygienist uses open-ended questions and reflective listening to
understand a hesitant patient's specific concerns before addressing them. This approach is an
example of:

# Options
- A) motivational interviewing, which builds patient-centered buy-in for the proposed plan (correct)
- B) informed refusal documentation
- C) the disease-control phase of treatment
- D) a periodontal reassessment

# Correct answer
A

# Rationale
Motivational interviewing uses open-ended questions, reflective listening, and eliciting the
patient's own reasons for change to strengthen engagement and case acceptance — a communication
approach well suited to presenting a plan to a hesitant patient.

**Why the distractors are wrong**
- B) Informed refusal documentation records a patient's decision to decline care; it does not
  describe this conversational technique.
- C) Disease-control phase refers to a stage of the treatment plan, not a communication technique.
- D) Periodontal reassessment is a clinical re-evaluation step, unrelated to this communication
  approach.
